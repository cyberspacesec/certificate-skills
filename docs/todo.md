## 🛠️ 技术栈

- **语言**: Go 1.23+
- **CLI框架**: cobra + viper
- **加密库**: crypto/x509, crypto/tls
- **网络库**: net/http
- **测试框架**: testing + testify
- **构建工具**: Make + goreleaser

## 📝 开发规范

1. **代码风格**: 遵循Go官方代码规范
2. **错误处理**: 使用标准的Go错误处理模式
3. **日志记录**: 使用结构化日志 (logrus/zap)
4. **配置管理**: 支持环境变量和配置文件
5. **文档**: 每个公开函数都要有注释

执行顺序建议：
1. 先完成基础架构 (阶段一)

3. 添加分析功能 (阶段三)
